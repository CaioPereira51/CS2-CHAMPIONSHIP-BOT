🎯 COMANDOS IMPLEMENTADOS

┌─────────────────────────────────────────────────────────────────────────┐
│ /criartime                                                              │
├─────────────────────────────────────────────────────────────────────────┤
│ Parâmetros:                                                             │
│   • nome (obrigatório)      - Nome do time                             │
│   • jogador1-5 (obrigatório) - 5 jogadores (mentions)                  │
│   • logo (opcional)          - URL da imagem                           │
│                                                                          │
│ Validações:                                                             │
│   ✅ Nome não vazio           ✅ Todos os 5 jogadores                   │
│   ✅ Sem duplicatas           ✅ Time único                             │
│   ✅ URL válida (se fornecida)                                          │
│                                                                          │
│ Resultado:                                                              │
│   📊 Embed bonito com cor azul (#0099ff)                               │
│   🖼️  Logo como thumbnail                                               │
│   👥 Lista de jogadores formatada                                       │
│   🕒 Timestamp automático                                               │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ /listartimes                                                            │
├─────────────────────────────────────────────────────────────────────────┤
│ Funcionalidades:                                                        │
│   • Lista todos os times cadastrados                                    │
│   • Um embed bonito para cada time                                      │
│   • Suporta até 10 times por mensagem                                   │
│   • Múltiplas mensagens se necessário                                   │
│                                                                          │
│ Mostra:                                                                 │
│   🏆 Nome do time             👥 Jogadores                              │
│   🖼️  Logo (se disponível)    📝 Quem criou                            │
│   🕒 Data de criação                                                    │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ /deletartime                                          🆕 COMANDO EXTRA  │
├─────────────────────────────────────────────────────────────────────────┤
│ Parâmetros:                                                             │
│   • nome (obrigatório) - Nome do time (com autocomplete)               │
│                                                                          │
│ Funcionalidades:                                                        │
│   🔍 Autocomplete inteligente (busca até 25 times)                     │
│   🗑️  Remove time do banco de dados                                     │
│   📊 Embed vermelho de confirmação                                      │
│   👥 Mostra jogadores do time deletado                                  │
└─────────────────────────────────────────────────────────────────────────┘

🚀 COMO USAR

  1. Criar bot no Discord Developer Portal
  2. Criar arquivo .env com:
     • TOKEN=seu_token_aqui
     • CLIENT_ID=seu_client_id_aqui
     • GUILD_ID=seu_guild_id_aqui
  3. Instalar: npm install
  4. Executar: npm start
  
  Leia SETUP_RAPIDO.md para detalhes!
